# 세계관 개요

괴력난신의 세계는 실제 고대사를 재현하지 않는다. 예맥, 사로국, 가락, 십제는 역사적 분위기와 문화 모티브를 빌린 가상의 팩션이며, 각 팩션은 정복전쟁의 초입에서 자신들의 패권을 세우려 한다.

첫 캠페인의 핵심 정서는 전면전 이전의 혼란이다. 성문이 닫히고, 군량이 징발되고, 떠돌이 병사와 밀정이 늘어나며, 피난민의 행렬이 길 위에 생겨난다. 괴이와 귀신은 전쟁의 바깥에 있는 존재가 아니라, 전쟁이 드러내는 이면이다.

## 핵심 질문

- 누가 먼저 정복전쟁의 명분을 만든 것인가?
- 성을 지키는 문지기는 백성을 지키는가, 왕을 지키는가?
- 전쟁의 뒤편에서 괴이, 귀신, 도깨비는 무엇을 먹고 커지는가?
- 철과 군량과 소문은 어떻게 왕보다 먼저 전장을 움직이는가?
