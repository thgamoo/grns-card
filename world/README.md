# 괴력난신 세계서

이 디렉토리는 카드 DB와 분리된 세계관 작업 공간이다.

현재 네 팩션은 실제 역사 국가가 아니라, 삼국시대와 원삼국시대의 문화적 모티브를 바탕으로 만든 가상 세력이다. 실제 인물, 사건, 국가를 직접 재현하지 않고 카드게임 캠페인에 필요한 지리, 전쟁, 전승, 단편 서사를 관리한다.

## 구조

- `maps/`: 캠페인 지도 원본, 내보낸 이미지, 지도 데이터
- `docs/`: 세계관 개요, 팩션, 장소, 연표, 확장팩 배경 문서
- `fiction/`: 단편 소설과 장면 조각
- `fiction/fragments/`: 카드 lore로 확장할 짧은 문장과 장면
- `data/`: 장소, 팩션, 사건을 카드 DB와 연결하기 위한 JSON 데이터

## 첫 작업 목표

1. 첫 캠페인 지역 지도 만들기
2. 정복전쟁의 발화점이 되는 변경 지대 정의
3. `드러나는 이면` 중립 카드 8장과 연결되는 사건 기록 작성
