# 괴력난신 문서

게임 규칙, 카드 설계, 웹사이트 운영 문서를 분리해 관리한다.

## 문서 목록

| 문서 | 내용 |
|---|---|
| [룰북](./rulebook.html) | 게임 목표, 준비 절차, 페이즈, 막사 이동, 전투, 돌파, 패배 조건 |
| [카드/클래스 설계](./card-design.html) | 세계관 축, 클래스 정체성, 카드 레이아웃, 스트럭쳐 덱 구성 |
| [카드 DB 운영 문서](./card-db-plan.html) | 로컬 실행, 배포, 데이터 버전 관리, 출력 기능 |

## 현재 문서 원칙

- 룰북은 플레이 중 참조할 수 있게 짧은 절과 명확한 용어를 우선한다.
- 카드/클래스 설계 문서는 밸런스와 카드 제작 방향을 기록한다.
- 카드 DB 운영 문서는 개발, 배포, 데이터 파일 관리만 다룬다.
