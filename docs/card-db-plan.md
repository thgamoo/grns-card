# 괴력난신 카드 DB 기획

## 개요

**괴력난신**은 원삼국~삼국시대의 어두운 신화적 상상력을 바탕으로 한 카드게임이다. 서양식 마법과 직업 체계 대신, 용력, 괴이, 반란, 귀신이라는 네 가지 초자연적 힘을 국가/세력별 클래스 정체성으로 삼는다.

첫 카드 DB 웹사이트는 다음 목적을 가진다.

- 핵심 클래스별 카드 색과 플레이 스타일을 한눈에 구분한다.
- 카드 타입, 종족, 비용, 효과, 플레이버 텍스트를 표준화한다.
- 카드 시리얼 넘버와 일러스트 이미지 필드를 유지한다.
- 예시카드를 통해 각 클래스의 덱 방향성을 확인할 수 있게 한다.
- 추후 실제 카드 데이터 JSON/API로 확장 가능한 구조를 유지한다.

## 로컬 실행

```bash
pnpm dev
```

개발 서버는 Node 내장 모듈만 사용하는 `server.mjs`로 실행된다. 별도 패키지 설치 없이 `pnpm dev`로 켤 수 있으며, 기본 URL은 `http://127.0.0.1:4173/index.html`이다.

## GitHub Pages 배포

`main` 브랜치에 push하면 `.github/workflows/pages.yml`이 실행되어 GitHub Pages에 배포한다.

```bash
npm run build
```

빌드 결과는 `dist`에 생성된다. 배포본에는 `window.GRNS_PUBLIC_MODE = true`가 주입되어 카드 편집기와 JSON Export 버튼이 빠지고, 검색/필터/PNG Export/A4 Print만 제공한다.

## 데이터 버전 관리

카드 데이터는 `./data` 디렉토리에 둔다.

- 버전 목록: `data/card-versions.json`
- 카드 DB 파일명 규칙: `data/card-[date-version].json`
- 예시: `data/card-2026-05-16-v1.json`

새 버전을 웹 편집기에서 export하면 `card-YYYY-MM-DD-vN.json` 파일로 다운로드된다. 그 파일을 `data` 디렉토리에 넣고, `data/card-versions.json`에 항목을 추가하면 상단 버전 드롭다운에서 선택할 수 있다.

브라우저 보안상 웹앱이 로컬 파일을 직접 덮어쓰지는 않는다. 개인용 DB 편집은 화면에서 수정한 뒤 JSON Export로 저장하는 흐름을 기준으로 한다.

## 카드 레이아웃

카드는 다음 구조를 기본으로 한다.

실물 출력 기준 카드 크기는 **60mm x 85mm**로 고정한다. 웹 미리보기와 프린트 뷰 모두 같은 `mm` 단위를 사용한다.

1. 상단
   - 왼쪽 위 비용: 원형 숫자
   - 카드명
   - 오른쪽 위 힘
   - 왼쪽 세로 띠: 클래스별 흑백 명도
2. 중단
   - 카드 일러스트: 현재는 빈 이미지 슬롯과 카드별 한자 `sigil`로 구분, 추후 이미지 경로를 `illustration` 필드에 입력
3. 하단
   - 효과 텍스트
   - 효과 비용: `[막: 1]` 형식
   - 카드 타입: 일반/트랩/제물을 큰 글씨로 표시
4. 외부 정보
   - 종족
   - 카드 타입
   - 힘

플레이버 텍스트는 현재 카드 표면에서 제외한다. 추후 텍스트 정책이 정해지면 JSON 필드로 다시 추가할 수 있다.

카드마다 `serial`, `sigil`, `illustration` 필드를 가진다. 현재 `illustration`은 빈 문자열로 유지하고, 카드 중앙에는 카드별 한자 `sigil`만 표시한다. 시리얼 넘버는 카드 하단에 작은 글씨로 표시한다.

클래스별 인쇄 구분은 다음을 기본으로 한다.

| 클래스 | 왼쪽 띠 | 오른쪽 아래 문양 |
|---|---|---|
| 고구려 · 맹혈전사 | white | △ |
| 진한 · 이매망량 | gray-200 | ★ |
| 가야 · 야철장 | gray-400 | ■ |
| 마한 · 주혼귀사 | gray-600 | ● |

## 출력과 이미지 내보내기

- 카드 선택 후 `PNG Export`를 누르면 해당 카드를 60mm x 85mm, 300dpi PNG로 저장한다.
- `A4 9장 Print`는 현재 필터/검색 결과를 기준으로 A4 한 장에 3열 x 3행, 총 9장씩 배치한다.
- 브라우저 인쇄 창에서 대상 프린터를 선택하거나, PDF로 저장하면 인쇄용 파일을 만들 수 있다.
- 60mm x 85mm 카드는 A4 세로 기준으로 3 x 3 배치가 물리적으로 적합하다.
- 카드 표면은 흰 배경과 검은 글씨 중심으로 구성해 흑백 프린트 환경에서도 읽히도록 한다.

## 카드 종류

### 일반유닛

전장에 배치되어 상대 일반유닛을 공격하거나 성의 문지기와 성주를 공성하는 유닛이다.

### 트랩유닛

가디언으로 배치될 경우 뒤집히며 효과를 발동하는 유닛이다. 상대가 공격할 때 패에서 비용만큼 막사 이동을 한 뒤 전장에 출정할 수 있다. 일반 유닛처럼 사용할 수도 있다. 대신 평균 힘은 낮다. 덱에는 최대 16장까지 넣을 수 있다.

## 스트럭쳐 덱 구성

스트럭쳐 덱은 클래스별 40장으로 구성한다. 같은 시리얼 넘버 카드는 최대 3장까지 넣을 수 있고, 트랩 카드는 최대 16장까지 넣을 수 있다.

각 클래스는 카드 종류 16종을 가진다.

| 구성 | 카드 종류 | 투입 매수 |
|---|---:|---|
| 일반 카드 | 5종 | 3, 3, 3, 3, 3 |
| 제물 카드 | 5종 | 2, 2, 2, 2, 2 |
| 트랩 카드 | 6종 | 3, 3, 3, 2, 2, 2 |

총합은 일반 15장, 제물 10장, 트랩 15장으로 총 40장이다.

### 제물유닛

제물 또는 자원으로 활용될 때 추가 효과를 발휘하는 유닛이다. 일반적으로 자원으로 사용할 경우 카드 한 장당 액션 하나를 소비하지만, 제물유닛은 추가 액션 수를 제공하거나 맨 위 자원일 때 필드 효과를 발휘한다.

## 필드 요소

### 패장

트래시/묘지 개념의 공개 카드 더미다. 기본적으로 더 이상 사용할 수 없는 카드들이 놓이지만, 마한의 귀신 카드는 이 영역을 적극적으로 참조한다.

### 막사

자원 더미 개념이다. 왼쪽막사와 오른쪽막사가 있다.

왼쪽막사는 성 안의 지휘실, 오른쪽막사는 야전 시 막사에 해당한다. 코스트를 소비할 때는 왼쪽막사 더미의 카드를 위에서부터 차례대로 오른쪽막사로 이동한다. 자신의 턴이 다시 돌아올 때, 플레이어는 오른쪽막사의 카드 순서를 정해 왼쪽막사에 다시 놓을 수 있다.

### 성

성은 플레이어의 생명선 같은 개념이다. 성은 가디언으로 구성되며, 가디언은 성주와 문지기를 말한다.

성주는 패장으로 이동하면 패배하는 카드다. 문지기는 성주 앞에 뒤집어져 있는 방어 카드다. 상대가 문지기를 공격 대상으로 지정하면 그 문지기를 뒤집는다. 뒤집은 카드가 트랩 타입이면 막사 이동 없이 효과를 발동한다. 트랩 타입이 아니면 뒤집은 뒤 곧바로 전투 처리를 한다.

### 군

덱 개념이다.

## 패배 조건

다음 중 하나를 만족하면 패배한다.

- 성주가 패장으로 이동한다.
- 군에 카드가 없다.

## 준비 절차

1. 카드 40장을 섞고 위에서부터 5장을 뽑는다. 이 5장은 가디언 후보가 된다. 마음에 들지 않는 경우 공개 후 전부 패장으로 이동시키고 다시 5장을 뽑는다.
2. 1장을 성주, 4장을 문지기로 선택한다. 성주는 뒤집어서 놓고, 성주 앞에 4개의 문지기를 뒤집어서 놓는다. 문지기는 일렬로 세우며 첫 번째 레인부터 동, 서, 남, 북으로 명명한다.
3. 군에서 5장을 뽑아 패를 준비한다. 가디언을 멀리건하지 않았다면, 처음 뽑은 패 5장을 공개 후 패장으로 이동시키고 다시 5장을 뽑는다.

## 메인페이즈 액션

메인페이즈에는 다음 행동을 할 수 있다.

- 출정하기: 패에서 전장으로 카드를 내려놓는다.
- 공격하기: 전장에 있는 카드로 상대 전장의 유닛이나 문지기를 공격한다.
- 희생하기: 막사의 카드 하나를 패장으로 보내고, 전장의 유닛 하나를 왼쪽막사 맨 아래에 놓는다.

이 게임에는 피해 누적이나 유닛 체력 개념이 없다. 카드가 공격하면 전투 결과에 따라 상대 카드를 한 번에 패장으로 보낼 수 있다.

## 상대 공격 시 행동

상대가 공격할 때, 손패의 트랩 타입 카드는 그 카드의 비용만큼 막사 이동을 한 뒤 전장에 출정할 수 있다.

## 전투 처리

전투는 공격 유닛의 힘과 방어자의 힘을 비교해 처리한다.

- 공격하는 유닛의 힘이 더 높으면 방어자의 유닛이 패장으로 이동한다.
- 서로 힘이 같으면 공격한 유저가 패를 한 장 버리고 방어자를 패장으로 보낼 수 있다.
- 공격하는 유닛의 힘이 더 낮으면 공격한 유닛이 패장으로 이동한다.

## 돌파

문지기가 아직 모두 처리되지 않은 상황에서도 돌파를 감행할 수 있다.

돌파를 할 때는 남은 문지기 1장당 패를 한 장씩 패장에 놓는다. 그렇게 하면 다른 문지기들을 무시하고 성주를 공격할 수 있다.

## 클래스별 디자인 방향

### 고구려 - 맹혈전사

고구려는 바바리안 컨셉의 공격적 클래스다. 용력은 몸 안에 깃든 괴력, 용혈, 전장의 분노를 의미한다.

- 핵심 키워드: 용력, 돌파, 분노, 선공, 자해 보상
- 추천 카드 타입: 일반유닛 중심
- 주요 플레이 감각: 빠르게 밀어붙이고, 돌파로 성주를 압박

### 진한(사로국) - 이매망량

진한은 도깨비와 이매망량 컨셉의 카운터 클래스다. 괴이는 상대의 행동에 반응해 나타나는 숨은 위협이다.

- 핵심 키워드: 트랩, 뒤집힘, 응답, 패에서 발동, 행동 취소
- 추천 카드 타입: 트랩유닛 중심
- 주요 플레이 감각: 상대 턴에 개입하고, 공격 선언과 카드 발동을 비틀며 템포 회수

### 가야 - 야철장

가야는 철기문화와 반란의 클래스다. 야철장은 제물을 태워 고코스트 유닛과 무기를 생산한다.

- 핵심 키워드: 제물, 주조, 막사, 고코스트, 반란
- 추천 카드 타입: 제물유닛과 대형 일반유닛
- 주요 플레이 감각: 초반 자원을 태워 중후반 강력한 철혈 유닛을 생산

### 마한 - 주혼귀사

마한은 귀신과 죽은 자의 힘을 다루는 클래스다. 귀신 카드는 패장과 파괴 타이밍에서 효과가 강해진다.

- 핵심 키워드: 패장, 파괴될 때, 되살림, 원혼, 저주
- 추천 카드 타입: 일반유닛과 제물유닛 혼합
- 주요 플레이 감각: 죽어도 손해가 적고, 패장에 쌓인 카드가 후반 자원이 됨
