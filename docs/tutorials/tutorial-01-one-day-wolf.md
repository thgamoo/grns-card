# 튜토리얼 1. 하룻 이리

## 목적

첫 번째 튜토리얼은 전투가 아니라 게임 시작 절차를 익히는 게임이다. 플레이어가 선후공 확인, 필드 요소 확인, 초기 4장 확인, 문지기/군영 선택, 첫 턴 2장 징집까지 이해하면 승리한다.

## 고정 설정

- 상대 성주: 하룻 이리 1/2
- 내 성주: 계시를 받은 범 4/4
- 선후공: 나 후공
- 승리 조건: 내 첫 턴 시작 시 2장 징집이 완료되면 하룻 이리가 패배한다.

## 진행 순서

1. 게임 시작 모달을 보여준다.
2. 시작 버튼을 누르면 코인플립 전에 필드 요소 투어를 진행한다.
3. 투어는 징집소, 군영, 문지기, 성주를 순서대로 설명한다.
4. 투어가 끝나면 코인플립 애니메이션을 보여주고, 고정 설정에 따라 나 후공을 표시한다.
5. 게임 화면을 어둡게 하고 징집소 위에서 카드 4장을 공개한다.
6. 공개된 4장 아래에 `문지기로`, `군영으로` 선택지를 보여준다.
7. `문지기로` 선택 시 공개 4장을 문지기로 가로 배치하고, 징집소 위에서 5장을 군영으로 가져온다.
8. `군영으로` 선택 시 1장을 더 뽑아 공개 4장과 함께 초기 군영으로 쓰고, 다음 4장을 문지기로 가로 배치한다.
9. 선택이 끝나면 `상대턴` 중앙 오버레이를 보여준다.
10. 상대 성주가 말풍선으로 `그르릉...`을 말한다.
11. 자동으로 `나의턴` 중앙 오버레이를 보여준다.
12. 내 턴 시작 처리로 징집소 위에서 2장을 군영으로 가져온다.
13. 2장 징집이 끝나면 승리 처리한다.
14. 승리 시 상대 성주가 산산조각 나는 연출을 보여준다.

## 구현 메모

- 1번 튜토리얼에서는 카드 선택으로 출정/공격까지 진행하지 않는다.
- 사용자의 선택은 초기 배치 선택지 두 개가 핵심이다.
- 로그는 32px 정사각형 아이콘 리스트로 표시한다.
- 로그 hover는 설명만, 클릭은 되돌리기 컨텍스트 창을 연다.
