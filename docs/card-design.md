# 갈래누리 카드/클래스 설계

## 세계관 축

**갈래누리**는 원삼국~삼국시대의 어두운 신화적 상상력을 바탕으로 한 카드게임이다. 서양식 마법과 직업 체계 대신, 용력, 괴이, 반란, 귀신이라는 네 가지 초자연적 힘을 국가/세력별 클래스 정체성으로 삼는다.

## 핵심 클래스

| 국가/세력 | 클래스 | 담당 테마 | 덱 정체성 |
|---|---|---|---|
| 예맥 | 개마무사 | 용력 | 높은 공격성, 돌파, 연속 공격, 전장 압박 |
| 사로국 | 이매망량 | 괴이 | 패트랩, 반응형 카운터, 뒤집힘 효과 |
| 가락 | 야철장 | 반란 | 책사 운용, 막사 조작, 고코스트 유닛 생산 |
| 십제 | 주혼귀사 | 귀신 | 패장 연계, 파괴될 때 효과, 되살림과 저주 |

## 카드 레이아웃

카드는 다음 구조를 기본으로 한다.

실물 출력 기준 카드 크기는 **63mm x 88mm**로 고정한다. TCG/포커 계열 표준 카드 규격에 맞추며, 웹 미리보기와 프린트 뷰 모두 같은 `mm` 단위를 사용한다.

1. 상단
   - 왼쪽 위 비용: 원형 숫자
   - 카드명
   - 오른쪽 위 힘
   - 왼쪽 세로 띠: 클래스별 흑백 명도
2. 중단
   - 카드 일러스트: 현재는 빈 이미지 슬롯과 카드별 한자 `sigil`로 구분, 추후 이미지 경로를 `illustration` 필드에 입력
3. 하단
   - 효과 텍스트
   - 효과 비용: `[막: 1]` 형식
   - 카드 타입: 일반/트랩/책사를 큰 글씨로 표시
4. 외부 정보
   - 종족
   - 카드 타입
   - 힘

플레이버 텍스트는 현재 카드 표면에서 제외한다. 추후 텍스트 정책이 정해지면 JSON 필드로 다시 추가할 수 있다.

카드마다 `serial`, `sigil`, `illustration` 필드를 가진다. 현재 `illustration`은 빈 문자열로 유지하고, 카드 중앙에는 카드별 한자 `sigil`만 표시한다. 시리얼 넘버는 카드 하단에 작은 글씨로 표시한다.

## 클래스별 인쇄 구분

| 클래스 | 왼쪽 띠 | 오른쪽 아래 문양 |
|---|---|---|
| 예맥 · 개마무사 | white | △ |
| 사로국 · 이매망량 | gray-200 | ★ |
| 가락 · 야철장 | gray-400 | ■ |
| 십제 · 주혼귀사 | gray-600 | ● |

## 스트럭쳐 덱 구성

스트럭쳐 덱은 클래스별 40장으로 구성한다. 같은 시리얼 넘버 카드는 최대 3장까지 넣을 수 있고, 트랩 카드는 최대 16장까지 넣을 수 있다.

각 클래스는 카드 종류 16종을 가진다.

| 구성 | 카드 종류 | 투입 매수 |
|---|---:|---|
| 일반 카드 | 5종 | 3, 3, 3, 3, 3 |
| 책사 카드 | 5종 | 2, 2, 2, 2, 2 |
| 트랩 카드 | 6종 | 3, 3, 3, 2, 2, 2 |

총합은 일반 15장, 책사 10장, 트랩 15장으로 총 40장이다.

## 클래스별 디자인 방향

### 예맥 - 개마무사

예맥은 개마무사의 돌격성과 산성 전투의 압박감을 앞세우는 공격적 클래스다. 용력은 몸 안에 깃든 괴력, 용혈, 전장의 분노를 의미한다.

- 핵심 키워드: 용력, 돌파, 분노, 선공, 자해 보상
- 추천 카드 타입: 일반 중심
- 주요 플레이 감각: 빠르게 밀어붙이고, 돌파로 성주를 압박

### 사로국 - 이매망량

사로국은 도깨비와 이매망량 컨셉의 카운터 클래스다. 괴이는 상대의 행동에 반응해 나타나는 숨은 위협이다.

- 핵심 키워드: 트랩, 뒤집힘, 응답, 패에서 발동, 행동 취소
- 추천 카드 타입: 트랩 중심
- 주요 플레이 감각: 상대 턴에 개입하고, 공격 선언과 카드 발동을 비틀며 템포 회수

### 가락 - 야철장

가락은 철기문화와 반란의 클래스다. 야철장은 막사를 운용해 고코스트 유닛과 무기를 생산한다.

- 핵심 키워드: 책사, 주조, 막사, 고코스트, 반란
- 추천 카드 타입: 책사와 대형 일반
- 주요 플레이 감각: 초반 자원을 태워 중후반 강력한 철혈 유닛을 생산

### 십제 - 주혼귀사

십제는 귀신과 죽은 자의 힘을 다루는 클래스다. 귀신 카드는 패장과 파괴 타이밍에서 효과가 강해진다.

- 핵심 키워드: 패장, 파괴될 때, 되살림, 원혼, 저주
- 추천 카드 타입: 일반과 책사 혼합
- 주요 플레이 감각: 죽어도 손해가 적고, 패장에 쌓인 카드가 후반 자원이 됨
