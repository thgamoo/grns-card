# 괴력난신 키워드 정리

이 문서는 카드 효과에 쓰는 대괄호 진언과 꺾쇄 규칙 용어를 정리한다. 룰북은 진행 순서를 설명하고, 이 문서는 카드 텍스트를 읽거나 작성할 때 참고하는 짧은 표로 사용한다.

기준은 `grns-card/src/content/rules.ts`와 최신 카드 데이터 `data/card-2026-05-28-v1`이다.

## 표기 원칙

- 진언 키워드는 카드 효과에서 `[]`로 감싼다. 예: `[출정]`, `[전투광]`
- 규칙 용어, 필드 위치, 종족, 카드명 참조는 `<>`로 감싼다. 예: `<성주>`, `<전진기지>`, `<짐승>`, `<괴유 - 정벌자>`
- 숫자나 X가 붙는 효과는 기본 키워드의 수치형으로 본다. 예: `[희생 X]`는 `[희생]`의 수치형이다.
- 힘을 낮추는 효과는 키워드로 쓰지 않고 `힘 -1`, `힘 -2`처럼 본문에 직접 적는다.
- `<전진기지>`는 필드 위치이고, `[전진기지]`는 그 위치 맨 위에 있을 때 적용되는 진언이다. `<후방기지>`와 `[후방기지]`도 같은 방식으로 구분한다.
- `매복`은 카드 타입이면서 진언으로도 쓰인다. 타입은 카드 정보의 `type`, 진언은 효과 안의 `[매복]`으로 구분한다.

## 진언 키워드

| 키워드 | 의미 | 메모 |
|---|---|---|
| `[출정]` | `<전장>`으로 배치될 때 효과를 이행한다. | 기본 등장 효과 |
| `[단말마]` | `<전장>` 혹은 `<성>`에서 `<매장지>`로 이동했을 때 효과를 이행한다. | 쓰러질 때 처리 |
| `[전진기지]` | `<전진기지>` 맨 위에 있을 때 항상 효과가 적용된다. | 위치 조건 진언 |
| `[후방기지]` | `<후방기지>` 맨 위에 있을 때 항상 효과가 적용된다. | 위치 조건 진언 |
| `[매복]` | 상대 공격 선언 시, 매복 선언 후 `<군영>`에서 `<전장>`에 배치하며 효과를 이행한다. | 타입명 `매복`과 구분 |
| `[자체보급]` | 해당 카드를 `<보급>`하여 효과를 발동한다. | 보급로 이동과 함께 처리 |
| `[상시]` | 조건을 만족하는 동안 지속적으로 효과를 적용한다. | 전장/위치 상태를 계속 본다 |
| `[전투광]` | 공격 선언 시 `<보급>`하지 않는다. | 공격적인 카드의 핵심 진언 |
| `[강화]` | 지정한 카드의 힘을 +X 한다. | 수치형으로 작성 가능 |
| `[공개]` | `<군영>`의 카드를 상대에게 공개한다. 단, 한 번 공개했던 카드는 다시 공개할 수 없다. | `[공개 1]` 표기 사용 중 |
| `[문지기 등장]` | `<비공개>` 상태에서 `<문지기>` 위치로 배치되었을 경우 효과를 이행한다. | 데이터에 `[문지기등장]` 1건 정리 필요 |
| `[퇴각]` | `<퇴각>`할 경우 효과를 이행한다. | 가락국 축적 흐름 |
| `[퇴출]` | `<군영>` 또는 `<징집소>`에서 `<검은 강>`으로 이동할 경우 효과를 이행한다. | `<매장지>`가 아니라 `<검은 강>`으로 간다 |
| `[성주등장]` | `<비공개>`인 상태에서 `<성주>`에 앞면으로 놓인 경우 효과를 이행한다. | 성주 첫 공개 시 처리 |
| `[왕살]` | `<문지기>`와 `<야전병>`을 무시하고 `<성주>`를 직접 공격할 수 있다. | 직접 성주 공격 |

## 수치형 진언

| 표기 | 의미 | 정리 방향 |
|---|---|---|
| `[희생 N]` | 지정된 위치에서 N장을 비용으로 보낸다. | 기본 키워드는 `[희생]` |
| `[희생 X]` | 카드 텍스트가 정한 X장만큼 비용으로 보낸다. | X의 기준을 같은 문장 안에 적는다 |
| `[공개 N]` | 조건에 맞는 카드 N장을 공개한다. | 기본 키워드는 `[공개]` |

## 규칙 용어

| 용어 | 의미 |
|---|---|
| `<선공>` | 전쟁 시작 시 먼저 턴을 진행하는 플레이어 |
| `<비공개>` | 카드를 뒷면으로 놓거나 손에서 공개하지 않아 상대가 정체를 알 수 없는 상태 |
| `<징집>` | `<징집소>`에서 카드를 뽑아 `<군영>`으로 가져오는 행동 |
| `<보급병>` | `<군영>`에서 `<후방기지>`로 내려놓아 비용으로 사용할 카드 |
| `<식량>` | 출정과 행동을 위해 지불하는 비용 |
| `<보급>` | `<후방기지>`의 카드를 `<전진기지>`로 옮겨 비용을 지불하는 처리 |
| `<대기>` | 아직 행동할 수 있는 준비 상태 |
| `<휴식>` | 이번 턴에 행동을 마친 상태. 정비 페이즈에 `<대기>`로 되돌아온다 |
| `<매복>` | 상대 공격 선언 시 매복 선언 후 배치하며 효과를 이행하는 진언 또는 카드 타입 |
| `<대가>` | 우회 공격, 이동, 생존 선언을 위해 카드를 `<군영>`이나 `<징집소>`에서 `<매장지>`로 보내는 행동 |
| `<퇴각>` | 전장의 카드를 `<후방기지>`로 돌려보내는 행동. 한 턴에 한 번만 가능 |
| `<퇴출>` | 지정된 카드를 `<검은 강>`으로 보내는 처리 |
| `<희생>` | 효과나 전투 처리를 위해 자신의 `<군영>`이나 `<징집소>`에서 카드를 `<매장지>`로 보내는 비용 |

## 필드 용어

| 용어 | 의미 |
|---|---|
| `<전장>` | 야전병 5장이 세로로 서는 구획. 공격, 이동, 장악 판단이 일어나는 전투 중심 |
| `<성>` | 문지기 4장과 성주 1장을 포함하는 구획 |
| `<야전병>` | 전장에 서는 카드. 공격, 이동, 장악 판단의 주체 |
| `<문지기>` | 성의 입구를 지키는 수비 카드. 비공개로 시작하고 공격받으면 등장한다 |
| `<성주>` | 성에 놓이는 핵심 카드. 성주가 매장지로 이동하면 즉시 패배한다 |
| `<진>` | 전장 안에서 야전병이 서는 자리 |
| `<군영>` | 손패. 출정, 보급병 배치, 퇴출과 공개가 이루어지는 시작점 |
| `<징집소>` | 덱. 전쟁 시작 시 40장으로 구성하고, 비면 즉시 패배한다 |
| `<후방기지>` | 아직 보급하지 않은 보급병이 머무는 곳 |
| `<전진기지>` | 보급에 사용된 카드가 이동하는 곳 |
| `<매장지>` | 쓰러진 카드가 가는 공개 더미. 성주가 이곳으로 가면 패배한다 |
| `<발각된 군영>` | 공개된 카드를 내려놓는 별도 공간 |
| `<검은 강>` | 퇴출로 죽은 카드들이 가는 별도 공간 |
| `<장악>` | 상대 문지기 위치가 비어 있을 때 보급 후 그 위치에 카드를 배치해 상대가 문지기를 다시 배치할 수 없도록 하는 행동 |

## 정리 필요 항목

- `[문지기등장]`은 `[문지기 등장]`으로 띄어쓰기 통일이 필요하다.
- `[공개 1]`은 `[공개]`의 수치형으로 유지할지, 문장형으로 풀어쓸지 결정이 필요하다.
- `[희생 N]`은 카드 효과 비용이고, 생존 선언에서 쓰는 비용은 `<대가>`로 쓴다.
- `[매복]`은 진언으로 쓰되, 카드 타입 `매복`과 혼동되지 않게 카드 작성 시 문맥을 분명히 한다.
- `<퇴출>`은 `<매장지>`가 아니라 `<검은 강>`으로 보내는 처리로 정리한다.
