# 갈래누리 키워드 정리

이 문서는 카드 효과에 대괄호로 적는 키워드와 규칙 용어를 정리한다. 룰북은 게임 진행 순서를 설명하고, 이 문서는 카드 텍스트를 읽을 때 필요한 짧은 참조표로 사용한다.

기준 데이터는 `data/card-2026-05-19-v1/cards`이다. 아래 정의는 현재 카드 텍스트에서 쓰이는 의미를 정리한 것으로, 룰북 확정 과정에서 세부 타이밍은 조정될 수 있다.

## 표기 원칙

- 키워드는 카드 효과에서 `[]`로 감싼다.
- 키워드와 숫자가 붙는 효과는 `[패 유실 1]`, `[약화 2]`처럼 한 덩어리로 적는다.
- 종족, 세력, 카드명 참조도 현재는 `[]`를 사용한다. 장기적으로는 키워드와 참조 표기를 구분할지 결정이 필요하다.
- 같은 효과는 같은 표기를 사용한다. 예를 들어 `야전 막사`와 `야전막사`는 하나로 통일한다.

## 기본 타이밍 키워드

| 키워드 | 의미 | 사용처 |
|---|---|---|
| `[출정]` | 이 카드가 출정했을 때 처리하는 효과. | 가장 기본적인 등장 효과 |
| `[단말마]` | 이 카드가 패장으로 이동할 때 처리하는 효과. | 십제/귀신, 사망 보상 |
| `[상시]` | 조건을 만족하는 동안 계속 적용되는 효과. | 사로국 약화 계열 |
| `[문지기 등장]` | 이 카드가 문지기로 놓였을 때 또는 문지기로 있는 동안 적용되는 효과. | 방어형 트랩 |
| `[야전막사이동]` | 이 카드가 야전막사로 이동할 때 처리하는 효과. | 막사 이동 보상 |
| `[지휘막사이동]` | 이 카드가 지휘막사로 이동할 때 처리하는 효과. | 막사 이동 보상 |

## 막사/위치 키워드

| 키워드 | 의미 | 메모 |
|---|---|---|
| `[야전막사]` | 카드가 야전막사에 있을 때 적용되거나, 야전막사를 참조하는 효과. | 위치 조건과 위치 참조가 섞여 쓰임 |
| `[지휘막사]` | 카드가 지휘막사에 있을 때 적용되거나, 지휘막사를 참조하는 효과. | 위치 조건과 위치 참조가 섞여 쓰임 |
| `[막사]` | 막사 전체를 가리키는 축약 표현. | 현재 1회만 사용됨. 통일 필요 |
| `[소도]` | 소도 영역 또는 소도에 있는 유닛을 참조한다. | 확장팩 전용 영역 |

## 상태 키워드

| 키워드 | 의미 | 메모 |
|---|---|---|
| `[대기]` | 전장에 놓였지만 즉시 전투/행동하기 어려운 상태. | 되살림, 교체 효과에서 사용 |
| `[정비]` | 전장에 놓였지만 준비가 필요한 상태. | `[대기]`와 차이를 룰북에서 확정 필요 |
| `[약화]` | 약화 계열 효과가 적용된 상태를 통칭한다. | `[약화] 된 모든 유닛`처럼 참조 |

## 전투 키워드

| 키워드 | 의미 | 메모 |
|---|---|---|
| `[전투광]` | 공격적인 전투 성향을 주는 키워드. | 상세 효과 확정 필요 |
| `[돌파]` | 문지기 방어를 넘어 성주 압박으로 이어지는 공격 키워드. | 상세 선언 절차 확정 필요 |
| `[왕살자]` | 성주를 직접 위협하는 고위 전투 키워드. | 과거 데이터에는 `[왕살]` 표기도 있음 |
| `[전사]` | 지정한 유닛을 전사 처리한다. 보통 패장 이동 또는 제거를 뜻한다. | 제거 효과의 표준어로 정리 필요 |

## 자원/손패/덱 키워드

| 키워드 | 의미 | 예시 |
|---|---|---|
| `[공개 1]` | 조건에 맞는 카드 1장을 공개한다. | 공개 후 드로우 보상 |
| `[패 유실 1]` | 패에서 카드 1장을 패장으로 이동한다. | 비용 또는 불이익 |
| `[패 유실 2]` | 패에서 카드 2장을 패장으로 이동한다. | 큰 불이익 |
| `[덱 유실 1]` | 덱 위에서 카드 1장을 패장으로 이동한다. | 덱 압박 |
| `[덱 유실 2]` | 덱 위에서 카드 2장을 패장으로 이동한다. | 덱 압박 |
| `[덱 유실 X]` | 조건으로 정한 X장만큼 덱에서 패장으로 이동한다. | X는 카드 텍스트에서 지정 |

## 약화 계열

| 키워드 | 의미 | 메모 |
|---|---|---|
| `[약화 1]` | 대상의 힘을 1 낮추는 약화 효과. | 대상 범위는 카드 텍스트가 지정 |
| `[약화 2]` | 대상의 힘을 2 낮추는 약화 효과. | 대상 범위는 카드 텍스트가 지정 |
| `[대약화 1]` | 상대 또는 상대 쪽 대상을 약화하는 효과로 사용된다. | 정확한 대상 범위 확정 필요 |
| `[전체약화 1]` | 전체 유닛 또는 넓은 범위에 약화 1을 적용한다. | 전체 범위 확정 필요 |
| `[성주약화 1]` | 성주에게 적용되는 약화 1. | 성주 전투력/방어값과 연결 |

| 키워드 | 주 세력 | 의미 |
|---|---|---|
| `[찬탈]` | 가락 | 상대나 막사 자원을 빼앗거나 되돌리는 반란 계열 효과. |
| `[희생]` | 가락 | 자기 카드를 비용으로 사용해 보상을 얻는 효과. |
| `[알]` | 가락 확장 | 알 유닛 또는 알 조건을 참조한다. |
| `[귀신]` | 십제 | 귀신 종족 또는 귀신 조건을 참조한다. |
| `[예맥]` | 예맥 | 예맥 세력 또는 예맥 조건을 참조한다. |
| `[인간]` | 공통 | 인간 종족 또는 인간 조건을 참조한다. |
| `[철기]` | 가락 | 철기 종족 또는 철기 조건을 참조한다. |

## 카드명 참조

현재 카드 효과에서 특정 카드명을 `[]`로 감싸 참조하는 경우가 있다.

| 표기 | 의미 |
|---|---|
| `[괴유 - 정벌자]` | 특정 카드명 참조 |
| `[마로 - 길잡이]` | 특정 카드명 참조 |
| `[드러난 우두머리 신]` | 특정 카드명 참조 |

## 정리 필요 항목

- `[왕살]`과 `[왕살자]` 중 하나로 통일한다. 최신 분리 데이터는 `[왕살자]`를 사용한다.
- `[패 유실 N]`과 `[덱 유실 N]`은 손패/덱에서 패장으로 이동하는 표준 처리로 고정하는 것이 좋다.
- `[대기]`와 `[정비]`는 둘 다 전장 배치 상태로 보이므로 행동 가능 여부, 공격 가능 시점, 전투 대상 가능 여부를 분리해 정의한다.
- `[야전막사]`, `[지휘막사]`는 위치 조건인지 위치 참조인지 카드 텍스트에서 맥락에 따라 다르다. 필요하면 `[야전막사 효과]` 같은 타이밍 키워드와 위치명 표기를 분리한다.
- 종족/세력/카드명 참조까지 대괄호를 계속 사용할지 결정한다. 카드 텍스트 가독성은 좋지만 자동 파싱에서는 키워드와 참조를 구분해야 한다.
